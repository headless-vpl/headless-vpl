# Showcase Roadmap

`showcaseItems` に含まれる 6 つの sample の現状と次の改良ポイントをまとめた一覧です。

## Flow Editor

- 現在: typed connector、palette、inspector、JSON export/import、drag edge 作成
- 次: minimap、subflow、edge reroute、port schema

## Block Editor

- 現在: Scratch-like layout、typed slot、C-block nesting、spec / support matrix
- 次: palette-driven block creation、variables / lists / custom block authoring、scene import/export

## Hybrid Editor

- 現在: stack snap と data edge の同居、inspector、scene persistence
- 次: stack relation の明示 serialization、typed data validation、mixed palette 拡張

## Blueprint Editor

- 現在: multi-pin graph、typed pin validation、inline value state、inspector
- 次: reroute node、exec/data lane の強調、macro graph

## State Machine

- 現在: transition editing、active state simulation、initial/final validation
- 次: guard / action、到達可能性検証、composite states

## Workflow Editor

- 現在: step palette、status metadata、JSON export/import、route editing
- 次: runtime trace、branch/error lane、sub-workflow library
